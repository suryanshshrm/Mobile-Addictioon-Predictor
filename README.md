# Mobile Addiction Prediction System

This project is a Machine Learning-based prediction system that analyzes smartphone usage behavior to determine mobile addiction levels. The model is trained using user activity data such as screen time, social media usage, night phone usage, and phone unlock frequency.

## Features
- Predicts mobile addiction levels
- Uses Decision Tree Classifier
- Trained on behavioral smartphone data
- Calculates model accuracy
- Beginner-friendly ML project

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Machine Learning

## Input Parameters
- Screen Time
- Social Media Usage
- Night Usage
- Unlock Frequency

## Machine Learning Algorithm
- Decision Tree Classifier

## Example Prediction
Input:
[9, 6, 1, 120]

Output:
Addicted

## Project Workflow
1. Load dataset using Pandas
2. Split dataset into training and testing data
3. Train Decision Tree model
4. Evaluate model accuracy
5. Predict addiction level from custom input
